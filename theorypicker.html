TheoryPicker

TheoryPicker is a tool to help those who are not behavioral scientists select a behavior change theory to guide health communication. For this project I was only given a bunch of content and a PowerPoint slide with how the client thought the site should be organized. 50 slides of content all linking to each other in a big confusing mess. I had to dive in head first, digest the content, and reorganize it for a more user friendly experience. I then created a few design mock ups for the client to approve before I started to build the site. I built it as a responsive site using Bootstrap as my framework. The only part I didn't build was the logic behind the actual picker tool. One of the developers on our team handled the javascript for that part. This project has a really small budget, so I was able to accomplish all of this in about a week and a half.
